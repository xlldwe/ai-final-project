# Звіт про чат-бот на Python з використанням бібліотеки TelebotAPI

## Вступ

У рамках проекту було розроблено чат-бот для Telegram з використанням бібліотеки TelebotAPI (PyTelegramBotAPI). Бот призначений для адміністрування інтернет-магазину одягу, з можливістю перегляду каталогу, оформлення замовлень та оплати. Включає функціонал для адміністраторів та користувачів.

## Ініціалізація проекту

- **Встановлення бібліотеки**: TelebotAPI встановлено у віртуальному середовищі.
- **API Token**: Отримано через BotFather.
- **Структура**: Код у `bot_telebot/` з обробниками, клавіатурами тощо.

## Реалізація базової структури бота

- **Команда /start**: Привітання та інструкції.
- **Інструкції**: Доступні команди після запуску.

## Обробка основних команд

- **/help**: Список команд.
- **/info**: Інформація про бота.
- **/catalog**: Перелік товарів.

## Створення інтерактивного каталогу товарів

- **Команда /catalog**: Список товарів з інлайн-кнопками.
- **Деталі товару**: При натисканні на кнопку - детальна інформація.

## Обробка замовлень

- **Оформлення**: Через інлайн-кнопки з підтвердженням.
- **Команда /order**: Надсилання замовлень адміністраторам.

## Реалізація меню для адміністраторів

- **Команда /admin**: Доступ для адміністраторів (за ID).
- **/add_item**: Додавання товарів.
- **/remove_item**: Видалення.
- **/orders**: Перегляд замовлень.

## Інтерактивне спілкування

- **Відповіді**: На повідомлення (/hello, подяка).
- **Автоматичні відповіді**: На питання про товари, замовлення.

## Реалізація клавіатури

- **Reply Keyboard**: Для команд /start, /catalog, /info.
- **Inline Keyboard**: Для товарів та замовлень.

## Функціонал для користувачів

- **Команда /feedback**: Залишення відгуків, надсилання адміністраторам.

## Валідація введених даних

- **Перевірка**: Ціни, команди, дані від користувачів.

## Реалізація системи оплати

- **Рахунок**: Створення для користувача.
- **Доставка**: Обробка.
- **Замовлення**: Попереднє, підтвердження/відміна.

## Завантаження бота на сервер

- **Сервер**: DigitalOcean, AWS тощо.
- **Встановлення**: Python, віртуальне середовище, пакети.
- **Запуск**: Копіювання файлів, запуск, автозапуск.

## Додаткові завдання

- **Логування**: Дії користувачів у лог.
- **Стан користувачів**: Збереження стану.
- **База даних**: SQLite для товарів та замовлень.
- **Мультимовність**: Підтримка мов.

## Результати проекту

Бот обробляє команди, управляє каталогом, приймає замовлення, адмініструє. Функціональний з валідованими даними, клавіатурою.

## Перевірка та тестування

Після розгортання протестовано функціонал, оплату, роботу на сервері.

## Висновки

Бот готовий, з усіма необхідними функціями. Код у `bot_telebot/`:
- `bot.py`: Головний.
- `handlers/`: user_handlers.py, admin_handlers.py.
- `keyboards/`: inline_keyboards.py, reply_keyboards.py.
- `database.py`: БД.
