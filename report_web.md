# Звіт про проектування веб-ресурсу з інтеграцією чат-ботів

## Опис проекту

Розроблено веб-сайт інтернет-магазину одягу з інтегрованим чат-ботом на основі GPT. Сайт включає головну сторінку, FAQ, блог, контактну форму. Чат-бот надає інформацію, відповідає на запитання, допомагає в навігації.

## Основні вимоги до проекту

### Функціональність веб-сайту
- **Головна сторінка**: Опис послуг, інформація.
- **FAQ**: Поширені запитання.
- **Блог**: Статті.
- **Контакти**: Форма зворотного зв'язку.
- **Інтеграція чат-бота**: На всіх сторінках.

### Функціональність чат-ботів
- **Відповіді на запитання**: Про продукти, оплату, доставку.
- **Інтерактивний інтерфейс**: Спілкування на сайті.
- **Контекст діалогу**: Запам'ятовування сесії.

## Технічні вимоги

### Реалізація фронтенту
- **HTML, CSS, JavaScript**: Для інтерфейсу.
- **Адаптивний дизайн**: Для мобільних.

### Функціональні можливості

#### Розробка інтерфейсу чат-бота
- **Кнопка запуску**: У правому нижньому кутку.
- **Вікно чату**: Поле введення, область відповідей.

#### Налаштування збереження даних
- **Сесія користувача**: Збереження контексту.
- **Збереження тексту**: Для аналізу.

#### Аналітика та покращення
- **Статистика**: Кількість запитів, теми.
- **Модуль аналітики**: Для відстеження.

## Пропозиції до функціоналу чат-бота

- **Відповіді на запитання**: Продукти, послуги, оплата, доставка, повернення.
- **Технічна підтримка**: Проблеми, навігація, реєстрація.
- **Рекомендації**: Персоналізовані для інтернет-магазину.
- **Мультимовність**: Підтримка мов через GPT API.

## Функціональне тестування

- **Перевірка**: Робота на сторінках, обробка запитів різної складності.
- **Мобільні**: Тестування на пристроях та браузерах.

## Висновки

Проект забезпечує повноцінний функціонал чат-бота для автоматизації підтримки, покращення взаємодії, зниження навантаження на команду.

## Структура сайту

- **index.html**: Головна.
- **blog.html**: Блог.
- **contact.html**: Контакти.
- **faq.html**: FAQ.
- **css/style.css**: Стилі.
- **js/chatbot.js**: Чат-бот.
- **js/main.js**: Основний JS.

Чат-бот підключається до Flask бекенду на /api/chat для відповідей через GPT.
